# Vben Admin 迁移检查清单

## 📋 迁移前准备

### 环境检查
- [ ] Node.js 版本 >= 18.0.0
- [ ] Git 已安装
- [ ] pnpm 已安装（`npm install -g pnpm`）
- [ ] 后端服务正常运行（端口 8080）

### 备份当前项目
- [ ] 备份 `lite-gateway-ui` 项目
- [ ] 记录当前项目的特殊配置
- [ ] 记录当前项目的自定义功能

---

## 🚀 第一阶段：项目初始化

### 1. 克隆 Vben Admin 项目
- [ ] 执行 `git clone https://github.com/vbenjs/vue-vben-admin.git lite-gateway-admin-vben`
- [ ] 进入项目目录 `cd lite-gateway-admin-vben`

### 2. 安装依赖
- [ ] 执行 `corepack enable`
- [ ] 执行 `pnpm install`（等待安装完成）
- [ ] 验证依赖安装成功

### 3. 启动测试
- [ ] 进入 `apps/web-antd` 目录
- [ ] 执行 `pnpm dev`
- [ ] 浏览器自动打开 `http://localhost:5173`
- [ ] 使用默认账号登录（admin/123456）
- [ ] 验证页面正常显示

---

## ⚙️ 第二阶段：项目配置

### 1. 环境变量配置
- [ ] 编辑 `apps/web-antd/.env.development`
- [ ] 设置 `VITE_GLOB_API_URL=/api`
- [ ] 设置 `VITE_GLOB_API_URL_PREFIX=/api`
- [ ] 设置 `VITE_GLOB_API_DEV_URL=http://localhost:8080`

### 2. Vite 代理配置
- [ ] 编辑 `apps/web-antd/vite.config.mts`
- [ ] 添加 `/api` 代理配置
- [ ] 设置代理目标为 `http://localhost:8080`
- [ ] 重启开发服务器验证配置

### 3. 请求拦截器配置
- [ ] 创建 `apps/web-antd/src/utils/request.ts`
- [ ] 配置请求拦截器（添加 Token）
- [ ] 配置响应拦截器（处理响应格式）
- [ ] 测试 API 请求是否正常
